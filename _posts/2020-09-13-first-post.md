## 1. HTML 태그 정리 ##
<br>
<br>
<br>

### 1. html ###
<br>
<br>
html 태그는 웹페이지를 코드로 표현하기 위해서 적는 태그이다.<br>
html 문서의 시작과 끝은  &lt;html&gt; &lt;/html&gt;로 나타낸다.<br>
<br>
<br>
&lt;!DOCTYPE html&gt;은 현재의 문서를 HTML문서로 지정하는 것을 의미한다.
<br>
<br>
<img width="482" alt="html태그" src="https://user-images.githubusercontent.com/62292136/93014131-562a0180-f5e9-11ea-8190-87ab8c3e5df8.PNG">
<br>
<br>
<br>
### 2. &lt;head&gt; ###
대개 웹페이지의 제목, 스타일, 링크 등을 표시한다.<br>
실제 웹페이지 화면에는 표시되지 않는다.<br>
<br>
<br>
<br>
### 3. &lt;title&gt; ###
문서의 제목과 웹페이지의 제목표시줄에 들어갈 내용을 기입한다.<br>
<br>
<br>
<br>
### 4. &lt;body&gt; ###
실질적인 웹브라우저에 표시되는 내용들을 적는다.<br>
<br>
<br>
<br>
### 5. &lt;div&gt; ###
문서의 내용들을 어떤 기준으로 묶을 때 사용한다.<br>
<br>
<br>
<br>
### 6. &lt;h&gt; ###
대개 문서에서 제목이 될만한 내용들을 적을 때 사용한다.<br>
&lt;h1&gt; &lt;/h1&gt; 에서 &lt;h6&gt; &lt;/h6&gt; 까지 존재한다.<br>
h1이 문자의 크기가 가장 크고 h6이 가장 작다.<br>
<br>
<br>
<br>
### 7. &lt;p&gt; ###
문서에서 단락을 구분할 때 사용한다.<br>
대개 긴 글을 표시할 때 자주 사용된다.<br>
<br>
<br>
<br>
### 8. &lt;ul&gt; ###
unordered list의 약자로 순서가 필요없는 목록들을 표시할 때 사용된다.<br>
<br>
<br>
<br>
### 9. &lt;ol&gt; ###
ordered list의 약자로 순서가 있는 목록들을 표시할 때 사용된다.<br>
<br>
<br>
<br>

### 10. &lt;li&gt; ###
&lt;ul&gt;과 &lt;ol&gt;을 사용했을 때 목록의 항목을 표시할 때 사용된다.<br>
<br>
<br>
<br>

### 11. &lt;a&gt; ###
링크를 추가할 때 사용된다.<br>
대개 &lt;a href = "URL"&gt; 내용 &lt;/a&gt;로 사용되며, <br>
url자리에 #을 넣으면 실제 링크가 아니라 링크역할만 하는 빈 링크를 표시할 때 사용된다.<br>
<br>
<br>
<br>

### 12. &lt;img&gt; ###
웹페이지에 이미지를 삽입할 때 사용된다.<br>
 &lt;img src = "이미지 경로&gt; 코드를 통해 사용한다.<br>
<br>
<br>
<br>
### 13. &lt;style&gt; ###
웹페이지에 적용할 CSS를 &lt;head&gt; &lt;/head&gt; 안에 표시한다.<br>
<br>
<br>
<br>
<br>
<br>
<br>

## 2. CSS 태그 정리 ##
<br>
<br>
<br>

### 1. font ###
해당 컨텐츠의 글꼴을 지정한다.<br>
<br>
<br>
<br>
### 2. color ###
해당 컨텐츠의 색상을 지정한다.<br>
<br>
<br>
<br>
### 3. background color ###
해당 컨텐츠의 배경색을 지정한다.<br>
<br>
<br>
<br>
### 4. margin ###
해당 컨텐츠의 여백(외부)을 지정한다.<br>
<br>
<br>
<br>
### 5. padding ###
해당 컨텐츠의 여백(태부)을 지정한다.<br>
<br>
<br>
<br>
### 6. width ###
해당 컨텐츠의 너비를 지정한다.<br>
<br>
<br>
<br>
### 8. height ###
해당 컨텐츠의 높이를 지정한다.<br>
<br>
<br>
<br>
### 9. radius ###
테두리를 둥글게 할 때 사용한다.<br>
<br>
<br>
<br>
### 10. position ###
해당 컨텐츠의 위치를 결정할 때 사용한다.<br>
<br>
<br>
<br>

### 11. float ###
해당 컨텐츠의 정렬 위치를 결정할 때 사용한다.<br>
<br>
<br>
<br>

### 12. text ###
해당 텍스트의 스타일을 지정할 때 사용한다.<br>
<br>
<br>
<br>

### 13. cursor ###
마우스 커서의 모양을 지정할 때 사용한다.<br>
<br>
<br>
<br>

### 14. overflow ###
해당 컨텐츠의 내용이 지정된 크기를 넘었을때 어떻게 처리할지 결정할 때 사용한다.<br>
<br>
<br>
<br>

### 15. border ###
해당 컨텐츠의 테두리에 스타일을 지정할 때 사용한다.<br>
<br>
<br>
<br>

### 16. a:hover ###
해당 컨텐츠에 마우스 포인터가 포커싱 됐을 때, 색상을 지정할 때 사용한다.<br>
<br>
<br>
<br>

### 17. list ###
목록 관련 태그들의 스타일을 지정할 때 사용한다.<br>
