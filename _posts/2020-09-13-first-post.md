### 1. html ###
<br>
<br>
html 태그는 웹페이지를 코드로 표현하기 위해서 적는 태그이다.<br>
html 문서의 시작과 끝은  &lt;html&gt; &lt;/html&gt;로 나타낸다.<br>
<br>
<br>

## 2. CSS 태그 정리 ##
<br>
<br>
<br>

### 1. font ###
해당 컨텐츠의 글꼴을 지정한다.<br>
<br>
<br>
<br>
### 2. color ###
해당 컨텐츠의 색상을 지정한다.<br>
<br>
<br>
<br>
### 3. background color ###
해당 컨텐츠의 배경색을 지정한다.<br>
<br>
<br>
<br>
### 4. margin ###
해당 컨텐츠의 여백(외부)을 지정한다.<br>
<br>
<br>
<br>
### 5. padding ###
해당 컨텐츠의 여백(태부)을 지정한다.<br>
<br>
<br>
<br>
### 6. width ###
해당 컨텐츠의 너비를 지정한다.<br>
<br>
<br>
<br>
### 8. height ###
해당 컨텐츠의 높이를 지정한다.<br>
<br>
<br>
<br>
### 9. radius ###
테두리를 둥글게 할 때 사용한다.<br>
<br>
<br>
<br>
### 10. position ###
해당 컨텐츠의 위치를 결정할 때 사용한다.<br>
<br>
<br>
<br>

### 11. float ###
해당 컨텐츠의 정렬 위치를 결정할 때 사용한다.<br>
<br>
<br>
<br>

### 12. text ###
해당 텍스트의 스타일을 지정할 때 사용한다.<br>
<br>
<br>
<br>

### 13. cursor ###
마우스 커서의 모양을 지정할 때 사용한다.<br>
<br>
<br>
<br>

### 14. overflow ###
해당 컨텐츠의 내용이 지정된 크기를 넘었을때 어떻게 처리할지 결정할 때 사용한다.<br>
<br>
<br>
<br>

### 15. border ###
해당 컨텐츠의 테두리에 스타일을 지정할 때 사용한다.<br>
<br>
<br>
<br>

### 16. a:hover ###
해당 컨텐츠에 마우스 포인터가 포커싱 됐을 때, 색상을 지정할 때 사용한다.<br>
<br>
<br>
<br>

### 17. list ###
목록 관련 태그들의 스타일을 지정할 때 사용한다.<br>
