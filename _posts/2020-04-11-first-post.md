## 2. 변경 사항에 대한 작업
### 1. add
파일을 staging area에 추가    
git remote add origin <url> reote repository를 등록     
     
### 2. mv
파일 이름 변경
### 3. reset
파일을 현재 버전으로 되돌리기 add무효화 변경된 staging area 무혀화, 가장 최근버전으로 돌아감
### 4. rm     
git과 로컬 디렉토리에서 모두 삭제
