## 1. pwd   
작업 중인 디렉토리의 절대 경로 표시  
사용방법 : pwd     
     
     
     
## 2. cd
작업 중인 디렉토리 이동     
./ -> 현재 디렉토리     
../ -> 상위 디렉토리     
/ -> 최상위 디렉토리     
사용방법 : cd [이동할 위치]     
![cd_pwd](https://user-images.githubusercontent.com/62292136/77247433-99c43400-6c74-11ea-9fdd-ffa3a8484243.PNG)     
     
     
     
## 3. ls
디렉토리와 파일 목록을 출력     
뒤에 -l을 붙이면 소유자 파일크기, 사용자 권한을 표시     
사용방법 : ls [디렉토리명 혹은 파일명]
