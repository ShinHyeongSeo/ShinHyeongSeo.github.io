## 웹 호스팅 실습 ##
<br/>
<br/>
<br/>
1) https://www.dothome.co.kr 주소로 들어가서 회원가입을 하고 무료 호스팅 신청을 한다.
<br/>
<br/>
<br/>
2) 일련의 정보들을 입력한 후 호스팅 신청을 완료한다.
<br/>
<br/>
<img width="949" alt="웹호스팅" src="https://user-images.githubusercontent.com/62292136/82051643-3ba45200-96f5-11ea-97c6-15137957ef26.PNG">     
<br/>
<br/>
<br/>
3) 부여받은 도메인 주소를 입력해서 Webpage가 호스팅 되었는지 확인한다.
<br/>
<br/>
<img width="960" alt="shinhyeongseo dothome co kr" src="https://user-images.githubusercontent.com/62292136/82051640-3a732500-96f5-11ea-94e7-8de744173454.PNG">     
<br/>    
<br/>
<br/>     
4) Filezilla 프로그램을 https://filezilla-project.org/ 사이트에서 다운로드한다.    
<br/>     
<br/>     
<br/>     
5) Filezilla 프로그램을 사용해 간단한 Webpage를 만든 후에 서버에 업로드한다.     
   (index.html을 만든 후 업로드)      
<br/>
<br/>
<img width="591" alt="index html_2" src="https://user-images.githubusercontent.com/62292136/82051636-38a96180-96f5-11ea-934c-5792092e6384.PNG">     
<br/>     
<br/>     
<br/>     
6) 마지막으로 잘되었는지 확인해주면 끝!
